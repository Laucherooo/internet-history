INITIATION

    1973,Robert Kahn and Vinton Cerf collaborated to develop a protocol that linked multiple networks together. This later became the Transmission Control Protocol/Internet Protocol (TCP/IP), a technology that linked multiple networks together. If one network was brought down, the others did not collapse.




    While working at Xerox, Robert Metcalfe developed a system using cables that allowed for transfer of more data over a network He named this system Alto Aloha, but later it wasn named as Ethernet. Over the next few years, Ted Nelson proposed hypertext to organize network information, and Unix became popular for TCP/IP networks. Dave Farber of the University of Delaware revealed a project to build an inexpensive network using dial-up phone lines. In 1982, the PhoneNet system was established and got connected to ARPANET and the first commercial network, Telenet.




    Ray Tomlinson came up with the idea while working for ARPANET. The government-funded research project that eventually became the internet. Tomlinson created a program that gave users the ability to send messages between connected computers on the ARPANET system. He put the two programs together, modified the code a little bit and thus the email program was born. He also gave the world the new icon for the information age, the “@” sign which was an obvious choice as it meant that there is some user at some location. The term email was coined several years later. The email became the killer app for ARPANET.
